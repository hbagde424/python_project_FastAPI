================================================================================
TROUBLESHOOTING
================================================================================

BACKEND WON'T START:
  1. Check Python version: python --version (should be 3.8+)
  2. Check virtual environment is activated
  3. Reinstall dependencies: pip install -r requirements.txt --force-reinstall
  4. Check port 8000 is not in use

FRONTEND WON'T START:
  1. Clear cache: rm -rf node_modules package-lock.json
  2. Reinstall: npm install
  3. Start: npm run dev
  4. Check port 3000 is not in use

API CONNECTION ERROR:
  1. Ensure backend is running: python run.py
  2. Check API URL in frontend/src/api/client.js
  3. Check CORS configuration in app/main.py
  4. Check browser console (F12) for errors

PORT ALREADY IN USE:
  Backend (8000):
    lsof -i :8000  # Find process
    kill -9 <PID>  # Kill process

  Frontend (3000):
    lsof -i :3000  # Find process
    kill -9 <PID>  # Kill process
